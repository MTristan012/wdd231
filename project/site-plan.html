Pokemon Info Cards - Site Plan

Site Name

Pokemon Info Cards
This name was chosen because the site will display detailed cards about individual Pokémon, making it clear and attractive to Pokémon fans and collectors.

Site Purpose

The purpose of this site is to provide detailed information cards about various Pokémon, including stats, types, evolutions, and images. It will serve as a simple and interactive Pokédex alternative for casual users and fans.

Scenarios

  • Which Pokémon has the highest base attack stat?
  • What does Eevee evolve into, and how?

Color Schema

Color primario: #ffcb05 – Used for accents, buttons, and highlights

Color secundario: #3b4cca – Used for headers, borders, and links

Typography

Fonts Used:

  • Press Start 2P – used for main headings to reflect the retro video game aesthetic.
  • Roboto – used for body text for readability and clean layout.
